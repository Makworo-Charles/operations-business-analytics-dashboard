# Data Dictionary

This document describes the fields used in the Operations & Business Analytics project.

| Column Name | Description |
|-----------|------------|
| Ticket_ID | Unique identifier for each operational ticket |
| Created_Date | Date and time when the ticket was created |
| Resolved_Date | Date and time when the ticket was resolved |
| Department | Business unit responsible for resolving the ticket |
| Issue_Category | Type of issue reported |
| Priority | Ticket priority level (Low, Medium, High, Critical) |
| Assigned_Agent | Agent assigned to resolve the ticket |
| SLA_Hours | Service Level Agreement resolution threshold |
| Cost_Per_Ticket | Estimated operational cost of handling the ticket |
| Resolution_Time_Hours | Actual time taken to resolve the ticket |
| SLA_Status | Indicates whether the ticket met or missed SLA |
| Month | Month extracted from Created_Date for trend analysis |
